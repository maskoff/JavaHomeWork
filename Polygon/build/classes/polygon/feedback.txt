Что понравилось:
 Алгоритм работает правильно .
 Код по сравнению многугольников вынесен в классы 
Что можно улучшить:
 Статические методы класса Poligon можно сделать обычными 
 К примеру: 
    public static ArrayList<Float> polygonArrayX (Polygon p)
 можно заменить на 
    public static ArrayList<Float> polygonArrayX ()
И в качестве данных данный метод будет использовать данные объекта . 
Обилие static методов - верный признак процедурного подхола и 
нарушение принципа инкапсуляции
